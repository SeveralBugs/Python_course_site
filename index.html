Работа с целыми числами

  1. Операция возведения в степень (**)
  2. Операция нахождения остатка (%)
  3. Операция целочисленного деления (//)
  4. Таблица приоритетов

Операция возведения в степень

Для возведения числа в степень используется оператор **. Пример вывода кода:

print(2 ** 0) # Ответ: 1 print(2 ** 1) # Ответ: 2 print(2 ** 2) # Ответ: 4 print(2 ** 3) # Ответ: 8

Операция нахождения остатка (%)

Оператор деления с остатком (%) возвращает остаток от деления двух целых чисел. Пример:

print(10 % 3) # Ответ: 1 print(10 % 4) # Ответ: 2 print(10 % 5) # Ответ: 0 print(10 % 6) # Ответ: 4

Операция целочисленного деления (//)

Оператор целочисленного деления (//) отбрасывает десятичную часть результата. Пример:

print(10 // 3) # Ответ: 3 print(10 // 4) # Ответ: 2 print(10 // 5) # Ответ: 2 print(10 // 6) # Ответ: 1

Таблица приоритетов

Оператор Описание
() Скобки
** Возведение в степень
- (унарный минус) Унарный минус
*, /, //, % Умножение, деление, целочисленное деление, остаток от деления
+, - Сложение, вычитание
